# Kaley's Chad-ZShell-Setup

## Install

### Dependencies

- `zsh` The Z shell.
- `exa` A ls replacement.

```
git clone https://github.com/DRAGONTOS/Chad-ZShell-Setup.git
cd Chad-ZShell-Setup
bash setup.sh
```
